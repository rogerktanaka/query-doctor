# Query Doctor

Query Doctor is an AI-powered SQL code review tool.

Instead of simply explaining SQL, Query Doctor reviews queries as if they were part of a professional Pull Request.

## Features

- SQL Review
- Performance Analysis
- Security Checks
- Readability Score
- Maintainability Score
- Best Practices
- AI Suggestions

## Status

🚧 Sprint #001 - MVP in development

## Vision

Help developers write better SQL in seconds.
